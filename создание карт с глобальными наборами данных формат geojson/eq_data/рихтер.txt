Землетрясения классифицируются по магнитуде на основании шкалы Рихтера. Файл содержит данные по всем землетрясениям
с магнитудой М1 и выше, произошедшим за последние 24 часа (на момент написания книги). Информация взята из одного
из каналов данных Геологического управления США, доступных по адресу: https://earthquake.usgs.gov/earthquakes/feed/
